Titanic Exploratory Data Analysis

Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, and factors affecting passenger survival.
The analysis includes data inspection, visualization, and interpretation of key variables such as gender, passenger class, age, and fare.

Objective
Understand dataset structure and missing values
Explore relationships between features and survival
Identify important factors influencing survival rate
Present insights using visualizations
